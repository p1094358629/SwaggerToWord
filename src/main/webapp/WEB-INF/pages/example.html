1. Overview

1.1. Version information

Version : 1.0

1.2. URI scheme

Host : 192.168.37.120:8080
BasePath : /

1.3. Tags

  • OrganController : 机构

  • PicMessageController : 图片验证码

  • RiskMenuController : 菜单查询控制器

  • RoleMenuController : 角色及菜单权限控制器

  • ShortMessageController : 短信验证码

  • UserAndRoleController : 用户权限

  • UserController : 用户信息

  • UserSecurityController : 用户安全

  • WebMessageController : 站内信

  • 机构管理 : 机构管理controller
